My Portfolio

Geographic Information Systems (GIS)

My GIS portfolio showcases projects involving spatial data analysis, mapping, and geographic visualization solutions. I specialize in using ArcGIS and QGIS for spatial analysis, mapping, and data visualization. When handling large-scale or repetitive datasets, I leverage Python to automate workflows, enhance data processing efficiency, and integrate advanced geospatial analysis techniques.

Spatial pattern analysis for environmental impact assessment

Remote Sensing

My remote sensing work focuses on satellite imagery analysis, spectral processing, and earth observation applications. I have utilized a range of technologies, including drones, LiDAR, and satellite imagery. The software I frequently use includes Agisoft, Pix4D, PCI Geomatics, Global Mapper, and Google Earth Engine. Specifically, I often integrate Google Earth Engine with Python for more detailed processing.

Multi-temporal satellite imagery for land cover change detection

Spectral signature analysis for vegetation health monitoring

LiDAR point cloud processing for 3D terrain modeling

Geodesy

My expertise in geodesy spans a wide range of projects, including land surveying, GNSS acquisition and processing, drone and LiDAR acquisition, hydrographic surveying, and earth modeling. I'm a licensed Mine Surveyor (Ministry of Energy and Mineral Resources, Indonesia) with an IHO Category A Certification in hydrographic surveying. I work with tools like Trimble Business Center, Leica Geo Office, Agisoft, Pix4D, Hypack, BeamworX, and programming in MATLAB and Python for earth modeling applications.

GNSS network design for high-precision positioning

Regional geoid model development for height system unification

Crustal deformation monitoring using integrated geodetic techniques

Advanced geodetic analysis for infrastructure monitoring

Reference frame establishment for national mapping projects
